## Tools

| Tool | Description |
|------|-------------|
| `generate_connection_request` | Generate a personalized connection request (max 300 chars) |
| `generate_inmessage` | Generate a professional InMail or direct message |
| `generate_post` | Generate an engaging LinkedIn post with hashtags |
| `analyze_profile` | Analyze a profile for actionable outreach insights |
| `generate_outreach_sequence` | Generate a 5-touch outreach sequence with timing |
| `generate_comment` | Generate an insightful comment for a LinkedIn post |
